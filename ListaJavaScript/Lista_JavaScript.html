quando utilizamos o JavaScript no VScode tudo que for um codigo JavaScript deve esta em duas tags <script>

var: variavel

window: O objeto window faz a manipulação das janelas do navegador. Sempre que abrirmos o navegador automaticamente esse objeto é gerado, isso acontece porque esse objeto representa exatamente essa janela que foi aberta.

prompt: prompt() exibe uma caixa de diálogo com uma mensagem opcional solicitando ao usuário a entrada de algum texto. Exemplo window.prompt('Qual é o seu nome?')

alert: O alert é uma funcionalidade essencial que permite aos desenvolvedores exibirem mensagens simples em caixas de diálogo no navegador.Ao utilizar a instrução alert() , é possível criar pop-ups informativos que podem conter mensagens de aviso, notificações ou até mesmo interações básicas com o usuário. Exemplo: Window.alert('Um virus foi detectado')

+: tem função de contatenação e não de adição, se você quiser fazer 5 + 5 em JavaScript e receber o valor 10 é necessario declarar esses numeros como Number pois até então eles são strings, se não a soma não será feita.

para ter um exmplo na pratica das tags citadas basta abrir o VScode e tirar o codigo do comentario

     
             var nome = window.prompt('Qual é o seu nome?') // Vai perguntar o nome...
             window.alert('É um grande prazer te conhecer, ' + nome + '!') // sinal de + tem função de contatenação

Temos duas formas de programar em JavaScript

A primira mais tracional usando + e ':

'O aluno '+ nome + ' de ' +idade+ ' anos tirou nota '+ nota+ '.'

E a segunda forma mais atual usando crase ` e ${}:

`O aluno ${nome} de  ${idade} anos tirou nota ${nota}.`
